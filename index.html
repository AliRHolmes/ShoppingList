🔧 Cloud Connection Setup

This app uses Google Sheets as a database. Follow these quick steps to connect it:

  1. Create a Google Spreadsheet.
  2. Go to Extensions > Apps Script, paste the companion code, and deploy as a Web App.
  3. Set executing to "Me" and access to "Anyone".
  4. Paste the Web App URL inside this HTML's GOOGLE_SCRIPT_URL variable (line 144).
